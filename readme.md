kys-cpp项目弄了个新格式，grp32。老的编辑器不兼容了，所以我写了个010editor的脚本，可以方便你改数据作弊

010editor是一个可以编程的十六进制编辑器

里面的注释010editor显示有问题
需要再设置里把template和hex editor里面的编码改为chinese


你的存档文件为kys-c++\game\save\r1.grp32到r10.grp32，分别对应第一个存档到第10个
